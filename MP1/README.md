# Mini Project 1 (MP1) — Star dataset

Standalone submission folder for HCDE 530 Mini Project 1.

## What is here

| File / folder | Purpose |
|---------------|---------|
| `mp1.ipynb` | Published analysis notebook (Sections 1–5) |
| `mp1.md` | Competency claim(s) |
| `stars.csv` | Source data (Kaggle star dataset) |
| `charts/` | Static PNG charts (also regenerated by the notebook) |
| `archive/legacy/` | Older matplotlib EDA drafts (not graded) |

## How to run

1. Open `MP1/` as the notebook working directory (VS Code / Jupyter).
2. Run **Kernel → Restart & Run All** on `mp1.ipynb`.
3. The first code cell installs dependencies: `jupyter`, `plotly`, `kaleido`, `pandas`.

From the repo root with a venv that already has those packages:

```bash
cd MP1
jupyter notebook mp1.ipynb
```

Chart exports require Kaleido (and its Chrome runtime for `write_image`).

## View on GitHub

Push to a **public** repo. GitHub renders `MP1/mp1.ipynb` in the browser; static chart images are also in `charts/` if Plotly cannot run in the viewer.
